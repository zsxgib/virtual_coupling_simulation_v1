# 论文 Figures 分析与改进建议

## 一、原论文 Fig.2-4 的作用分析（IEEE TNNLS 2023）

| 图 | 核心作用 | 回答的问题 |
|----|----------|------------|
| Fig.2 | 共识误差收敛 | 误差能收敛到0吗？ |
| Fig.3 | 内部状态误差 | 状态会振荡后收敛吗？ |
| Fig.4 | 耦合权重演化 | 参数如何从0增加到稳定？ |

**特点**: 每张图只回答一个问题，非常纯粹。

## 三、改进建议

### 1. 保留
- **Fig.2**: 误差收敛对比（证明控制性能不下降）
- **Fig.5**: 通信量柱状图（证明通信效率提升）

### 2. 改进 Fig.3（触发事件 → 触发间隔分布）

**原问题**: 触发怎么随时间分布？

**改进方案**: 改成触发间隔分布图

- X轴: 间隔区间（0-0.1s, 0.1-0.5s, 0.5-1s, 1s+）
- Y轴: 该区间的触发次数
- 作用: 回答"有Zeno问题吗？"

### 3. 改进 Fig.4（阈值演化）

**原问题**: 阈值怎么随时间变化？

**当前问题**: 三条曲线太相似

**改进方案**:

- 4a: 保留阈值演化（但突出方案差异）
- 4b: 参数估计可删除或简化

### 4. 新增 Fig.3 替代方案（按时间段的通信量拆分）

把总通信量按时间段拆分：

| 子图 | 时间段 | 作用 |
|------|--------|------|
| (a) | 0-10s | 前期触发频繁 |
| (b) | 10-30s | 中期逐渐减少 |
| (c) | 30-60s | 后期稳定触发少 |

**作用**: 一眼看出"前期多、后期少"的趋势

## 四、画图原则总结

| 原则 | 说明 |
|------|------|
| 一图一问题 | 每张图只回答一个核心问题 |
| 对比优先 | 用对比而非叠加展示差异 |
| 直观优先 | 能用柱状图就不用散点图 |
| 删繁就简 | 可选内容移到附录 |
